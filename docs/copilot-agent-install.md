# Guía: instalar y seleccionar un Custom Agent (Awesome Copilot)

Estos pasos muestran cómo instalar y usar un agente personalizado (chat mode) desde la colección "Awesome Copilot" en Visual Studio Code.

1. Abrir Visual Studio Code.
2. Abrir el panel de GitHub Copilot / Copilot Chat (barra lateral).
3. Buscar el selector de agentes/chat modes (icono o menú desplegable en la parte superior del panel de chat).
4. Hacer clic en "Browse agents" o "Discover agents".
5. En la búsqueda, escribir "Awesome Copilot" y localizar el agente deseado (por ejemplo "Awesome Copilot — Focused Dev Agent").
6. Hacer clic en "Install" o "Add" y seguir las indicaciones (posible recarga de la ventana).
7. Tras la instalación, volver al panel de Copilot Chat y seleccionar el agente instalado en el selector de modos.
8. Iniciar una conversación y pedir tareas específicas (por ejemplo: "Genera pruebas unitarias para `script.js`" o "Sugiere una refactorización minimal para el módulo X").

Notas y permisos:
- Algunos agentes pueden requerir permisos adicionales o configuración (acceso a repositorios, tokens). Revisar y aprobar conscientemente.
- No instalar agentes de fuentes no confiables. Preferir agentes publicados por autores conocidos o verificados.

Si quieres, puedo crear un archivo de ejemplo de configuración del agente o añadir una entrada en `README.md` con un enlace directo al agente recomendado.
